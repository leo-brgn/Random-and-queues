Initialisation

Crer un projet R :

File -> New Project... -> New Directory

-> New Directory -> New Project (type du projet)

-> Renseigner le nom du projet et son emplacement

Un objet R Project va apparatre dans le dossier spcifi. Placer vos fichiers avec le code initial (generateur.R et seance1.R) dans le mme dossier.
